# Сымбат — сайт эксперта

Сайт бухгалтера-аналитика и налогового консультанта **Сымбат** для малого и среднего бизнеса Казахстана.

Позиционирование: не массовый онлайн-бухгалтер, а бутиковый эксперт по диагностике учёта и снижению налоговых рисков. Фирменный продукт — «Паспорт безопасного перехода ИП → ТОО».

## Стек

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Деплой: Vercel
